Тест Booking API

Интерактивная страница для тестирования API бронирования поездок

Инструкция по использованию:

  • Введите действительный JWT токен в поле выше для аутентификации
  • Убедитесь, что сервер запущен по указанному базовому URL
  • Для каждого типа запроса заполните необходимые поля и нажмите соответствующую кнопку
  • Результаты запроса будут отображены в области ответа

Примеры запросов:

  • Создание бронирования: PUT /api/booking с JSON { "tripId": 123, "message": "Привет, можно присоединиться?" }
  • Получить бронирования: GET /api/booking/123
  • Получить сообщения: GET /api/booking/messages/abcdef
  • Отправить сообщение: PUT /api/booking/messages с JSON { "bookingId": "abcdef", "content": "Во сколько выезжаем?" }
